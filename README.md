## üßπ  Data Cleaning 
 load data in Power BI:

Click Transform Data ‚Üí it opens Power Query Editor.

Common Cleaning Steps:

Rename Columns ‚Üí make them clear and consistent.

Change Data Types ‚Üí set columns to correct types (Date, Decimal, Whole Number, Text).

Remove Unnecessary Columns ‚Üí right-click column ‚Üí Remove.

Remove Blank or Error Rows:

Home ‚Üí Remove Rows ‚Üí Remove Blank Rows / Remove Errors.

Handle Missing Values (Nulls):

Replace with a fixed value ‚Üí Transform ‚Üí Replace Values.

Or remove rows with nulls.

Format Text Columns:

Transform ‚Üí Format ‚Üí Trim, Clean, UPPER/lower case.
